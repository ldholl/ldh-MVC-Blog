# ldh-MVC-Blog

## About

- A model-view-controller site that allows users to sign up, make, edit and delete posts, and comment on other user's posts. 


## Technologies
- HTML
- CSS
- Javascript

- Handlebars
- Bulma.io

- Express
- Express-Sessions
- Bcrypt
- MySQL2
- Sequelize
